# BlogGPT: Автоматизированный генератор постов с OpenAI и NewsAPI

## О проекте

BlogGPT — это Python-проект для автоматического создания тематических постов на основе свежих новостей и искусственного интеллекта (OpenAI GPT). Решение идеально подходит для блогов, Telegram-каналов, образовательных целей, интеграций с Zapier, а также тех, кто хочет быстро получать уникальный контент с SEO-метаданными.

## Основные функции

- Получение свежих новостей по выбранной теме с NewsAPI.
- Генерация полного поста, title и SEO-метаданных с помощью OpenAI GPT-4/3.5.
- REST API для программной генерации постов, реализованное на FastAPI.
- Готовая интеграция с Telegram и Zapier (через вебхуки).
- Простая настройка деплоя на Koyeb, Render, Heroku, Google Cloud Run, Vercel и др..

## Пример архитектуры

```
[Запрос темы] → [Получение новостей] → [Генерация текста, title и мета] → [Возврат JSON]
```

## Быстрый старт

1. Клонируйте репозиторий:
   ```
   gh repo clone ZerocoderUniversity/AI-course-prompt-engineering
   cd bloggpt
   ```
2. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
   Минимальный requirements.txt:
   ```
   fastapi
   uvicorn
   pydantic
   openai==0.27.0
   requests
   httpx
   tiktoken
   ```

3. Создайте файл `.env` и укажите переменные:
   ```
   OPENAI_API_KEY=ваш_ключ_OpenAI
   NEWSAPI_KEY=ваш_ключ_NewsAPI
   ```

4. Запустите локальный сервер:
   ```
   uvicorn app:app --host 0.0.0.0 --port 8000
   ```

## Описание API

### POST /generate-post

- **Описание:** Генерирует тематический пост на основе свежих новостей и текстов GPT.
- **Параметры запроса:** JSON, например:
  ```json
  {
    "topic": "Python programming"
  }
  ```
- **Ответ:**
  ```json
  {
    "title": "Оказывается, Python вновь стал самым популярным языком",
    "metadescription": "Новости и тренды Python на сегодня.",
    "postcontent": "...сгенерированный текст поста..."
  }
  ```

- **Проверка работоспособности:**  
  GET /heartbeat → `{"status": "OK"}`

## Интеграция и автоматизация

- **Telegram**: Можно использовать через Zapier, настроив связку для отправки сообщений по расписанию или по API-триггеру.
- **Zapier**: Инструкция по созданию Zap — получение поста → отправка в Telegram-канал.

## Деплой/Hosting

### Koyeb

1. Создайте новый сервис (Web Service) и подключите репозиторий GitHub с проектом.
2. Запустите команду: `python app.py`
3. В настройках задайте переменные окружения:
   - `OPENAI_API_KEY`
   - `NEWSAPI_KEY`
4. После деплоя получите публичный URL для использования API.

### Render

1. Зарегистрируйтесь на [Render.com](https://render.com).
2. Создайте сервис и импортируйте проект с GitHub.
3. Укажите команду запуска:  
   `uvicorn app:app --host 0.0.0.0 --port 8000`
4. Добавьте переменные окружения:  
   - `OPENAI_API_KEY`
   - `NEWSAPI_KEY`
5. Запустите сервис.
