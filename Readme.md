# BlogGPT: Автоматизированный генератор постов с OpenAI

## О проекте
Это учебный проект из курса Prompt Engineering от ZeroCoder, использует часть образовательного контента курса.
Проект предназначен для автоматического создания постов с заданной тематикой с использованием искусственного интеллекта (OpenAI GPT). 

## Основные функции
- Генерация текста поста, заголовка и SEO-метаданных с помощью OpenAI GPT.
- REST API для программной генерации постов, реализованное на FastAPI.
- Готовая интеграция с Telegram и Make/Zapier (через вебхуки).
- Простая настройка деплоя на Koyeb, Render, Heroku, Google Cloud Run, Vercel и др..

## Пример архитектуры
```
[Запрос темы] → [Генерация текста, title и мета] → [Возврат JSON]
```

## Описание API

### POST /generate-post

- **Описание:** Генерирует тематический пост.
- **Параметры запроса:** JSON, например:
  ```json
  {
    "topic": "Python programming"
  }
  ```
- **Ответ:**
  ```json
  {
    "title": "Оказывается, Python вновь стал самым популярным языком",
    "metadescription": "Новости и тренды Python на сегодня.",
    "postcontent": "...сгенерированный текст поста..."
  }
  ```

## Используемые технологии
- Python 3.10+
- FastAPI
- OpenAI GPT API
- OAuth2 токены и переменные окружения

## Применение
- Генерация контента для блогов и соцсетей
- Быстрая генерация SEO-описаний и заголовков для статей
